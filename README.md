# LevelUp2018Prototype
## Investigation of IoT platforms
[Mainflux](https://www.mainflux.com/), [Thinger.io](https://thinger.io/), [Thingsboard](https://thingsboard.io/) and [Devicehive](https://www.devicehive.com/) of which Devicehive were choosen due to being the only one with server code available.
Two different devices where connected to the Devicehive platform: ESP8266 and Raspberry Pi 3. 
The tutorials ([Chip in the cloud](https://docs.devicehive.com/v2.0/docs/chip-in-the-cloud-esp8266) and [Raspberry Pi and Temperature Sensor](https://docs.devicehive.com/v2.0/docs/raspberry-pi-and-temperature-sensor#section-create-client-app) )found in the Devicehive documentation where used to connect this devices to the cloud, using the free devicehive server PlayGround.
